## How to connect Gainium to Kucoin

**1. Log in to Kucoin and go to the API Keys Page**

Once you have logged into your Kucoin account, hover on “My Account” on the top right and click on API management. (Kucoin allows the creation of a subaccount, to create API for the subaccount, you need to select the subaccount first).

Then click on API.
![Screenshot 2023-03-30 at 19.38.53.png](https://content.gainium.io/images/content/help/Screenshot_2023_03_30_at_19_38_53_2251ade3b5-5b0c25.webp)
*Login*

**2. Click on the “Create API” green button**
![Screenshot 2023-03-30 at 19.39.13.png](https://content.gainium.io/images/content/help/Screenshot_2023_03_30_at_19_39_13_17c7bf0238-fd4d2d.webp)
*Api Management page*

**3. Fill in the Create API page**
- Complete the “API Name” and “API Passphrase” fields- The following section is greyed out by default- Tick the "Trade" option - Tick the “IP Restriction”, (please read the documentation regarding API Key security [here](https://gainium.io/help/enhancing-security), where you can also find the IP addresses to fill in the form)
- Click “Next” and complete the image caption popup
![Screenshot 2023-03-30 at 19.41.44.png](https://content.gainium.io/images/content/help/Screenshot_2023_03_30_at_19_41_44_1e3e8a1c9b-7738b9.webp)
*Create API*



**5. Complete the email verification**

Once you have completed step 4, you will need to activate your API by clicking “Activate API” from your mailbox.
![Screenshot 2023-03-30 at 19.44.11.png](https://content.gainium.io/images/content/help/Screenshot_2023_03_30_at_19_44_11_f331376de9-27be41.webp)
*Verification*
![Screenshot 2023-03-30 at 19.45.02.png](https://content.gainium.io/images/content/help/Screenshot_2023_03_30_at_19_45_02_632f6399ee-deec52.webp)
*Activate API*

**6. Copy the Key and Secret Key**

Once you have passed the email security verification, you will receive your keys.

Store the API secret in a safe place, as you will only see them on the screen once at this stage.
![Screenshot 2023-03-30 at 19.45.29.png](https://content.gainium.io/images/content/help/Screenshot_2023_03_30_at_19_45_29_abff449388-a009f6.webp)
*API Created*



**7. Paste the keys in Gainium**
- Open the [Exchanges](https://app.gainium.io/exchanges) page in Gainium and click on the “Add New” orange tab at the top of the page.-  Fill in the information in the Add New exchange popup.
- Click on the “Add Exchange” orange tab
![Screenshot 2023-03-30 at 19.28.37.png](https://content.gainium.io/images/content/help/Screenshot_2023_03_30_at_19_28_37_0bd61e6261-d30e7d.webp)![Screenshot 2023-03-30 at 19.48.41.png](https://content.gainium.io/images/content/help/Screenshot_2023_03_30_at_19_48_41_025d218ce8-c82131.webp)
*Add New Exchange*

You are all set! The newly added exchange will now show up.
